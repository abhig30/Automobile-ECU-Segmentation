# Automobile ECU Segmentation

This project demonstrates memory segmentation and multilevel segmentation
used in an Automobile Engine Control Unit (ECU).

## Features
- Code, Data, Stack, and I/O Segmentation
- Multilevel Segmentation in Data Segment
- Logical to Physical Address Translation
- Segmentation Fault Detection

## Technologies Used
- HTML
- CSS
- JavaScript
